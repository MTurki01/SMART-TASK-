# متابع المهام

موقع عربي لإدارة المهام الشخصية والوظيفية باستخدام HTML وCSS وJavaScript، مع تصميم مناسب للهاتف وتخزين محلي، ويمكن ربطه اختياريًا مع Firebase Realtime Database بدون تسجيل دخول.

## الملفات

- `index.html`: صفحة الموقع الرئيسية وإعدادات Firebase.
- `styles.css`: تصميم الواجهة العربية المتجاوبة.
- `app.js`: منطق المهام، الحفظ المحلي، المزامنة الاختيارية مع Firebase، الصوت، واستخلاص النص من الصور.

## النشر على GitHub Pages

1. أنشئ Repository جديد في GitHub.
2. ارفع الملفات الثلاثة الموجودة هنا إلى جذر المستودع:
   - `index.html`
   - `styles.css`
   - `app.js`
3. من إعدادات المستودع افتح:
   `Settings` ثم `Pages`.
4. اختر:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. احفظ الإعدادات وانتظر حتى يظهر رابط الموقع.

## إعداد Firebase بدون تسجيل دخول

في Firebase Console:

1. افتح مشروعك `task-27345`.
2. من `Realtime Database` تأكد أن قاعدة البيانات مفعلة.
3. لأن التطبيق يعمل بدون تسجيل دخول، استخدم قواعد عامة فقط إذا كنت تريد مزامنة نفس المهام لكل من يفتح الموقع:

```json
{
  "rules": {
    "publicTaskflow": {
      ".read": true,
      ".write": true
    }
  }
}
```

تنبيه: هذه القواعد تجعل البيانات عامة لأي شخص يعرف رابط قاعدة البيانات. إذا كنت تريد خصوصية حقيقية لكل مستخدم، يجب إعادة تفعيل تسجيل الدخول أو استخدام Backend.

## ملاحظة مهمة عن إخفاء الكود

إذا نشرت الموقع كـ HTML/CSS/JS، فالمستخدم يستطيع رؤية كود الواجهة من المتصفح. يمكنك جعل Repository خاصًا، لكن كود الواجهة المنشورة سيظل قابلًا للعرض لمن يفتح الموقع. إذا أردت إخفاء منطق حساس، يجب نقله إلى Backend أو Cloud Functions.
